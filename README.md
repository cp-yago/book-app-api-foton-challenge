### Demonstration

### Run local


- Clone this repo

```bash
$ git clone https://github.com/cp-yago/book-app-api-foton-challenge.git
```

- Install dependencies

```bash
# go to project folder
$ cd book-app-api-foton-challenge

# install dependencies using yarn
$ yarn
```

- Run tests

```bash
$ yarn test
```

- Run project

```bash
$ yarn start
```
